# WEDDING_PHOTOGRAPHER_FINDER
This single-page tool helps couples compare two wedding/portrait photos from different studios and produces a clear, decision-ready recommendation.
Studio Photo Comparison — Professional Wedding Image Analyzer
This single-page tool helps couples compare two wedding/portrait photos from different studios and produces a clear, decision-ready recommendation.
Instead of vague opinions, it generates a structured evaluation using practical, industry-aligned signals such as:
Face/subject focus sharpness (is the most important area actually crisp?)
Subject separation / depth feel (does the couple pop from the background?)
Composition discipline (balance, focal placement, storytelling structure)
Color consistency & grading (natural tones vs heavy preset/filter look)
Creativity signals (custom vision vs “generic Instagram pose” vibes)
Timelessness / album value (will it still feel meaningful years later?)
Trend risk (over-filtered looks that may age quickly)
Output
After uploading both images, the tool provides:
A score for each photo
A winner recommendation
A professional explanation in plain language (why this studio is the better choice)
Privacy
All analysis runs locally in your browser. Images are not uploaded anywhere.
How to use
Open index.html (locally or via GitHub Pages)
Upload Studio 1 image
Upload Studio 2 image
Click Analyze & Generate Decision
